### Installing

Just git clone the repo

    sudo git clone https://github.com/suraei/hopSSL.git


Give execution permisions to the script

    sudo chmod +x ./hopSSL

And run the script, u can use a URL or a Domain list

    ./hopSSL -t <URL>|<list>
